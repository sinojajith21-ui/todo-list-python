# To-Do List App (Python)

A simple command-line To-Do List Manager that allows you to:

- Add tasks
- View tasks
- Save tasks to a file (`tasks.txt`)
- Persistent storage (tasks stay saved even after closing the program)

### How It Works
- Tasks are added via the menu.
- All tasks are stored inside a file called `tasks.txt`.
- When you reopen the program, your previous tasks remain.
